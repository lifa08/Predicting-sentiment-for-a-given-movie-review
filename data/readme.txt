This folder should be filled with 4 data folders:
1.) aclImdb: You can download this from http://ai.stanford.edu/~amaas/data/sentiment/
2.) gensim: Contents inside this folder are created from aclImbd by running the file notebooks/pre_1_2_3_Wemb_gensim.ipynb
3.) imdb: Contents inside this folder are created from aclImbd by running the file notebooks/pre_3_aclImdb_2idx.ipynb
4.) info_word2vec_vs_gensim: Contents for this folder are already included, so no action required.
